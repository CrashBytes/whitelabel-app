# Default Assets

Place the following image files in this directory:

## Required Files

### icon.png
- **Size**: 1024x1024px
- **Format**: PNG with transparency
- **Purpose**: App icon for iOS and Android
- **Guidelines**: Simple, recognizable design that works at small sizes

### splash.png
- **Size**: 1242x2436px (iPhone X resolution)
- **Format**: PNG
- **Purpose**: Splash screen shown when app launches
- **Guidelines**: Center your logo/branding, account for safe areas

### adaptive-icon.png
- **Size**: 1024x1024px
- **Format**: PNG with transparency
- **Purpose**: Android adaptive icon foreground
- **Guidelines**: Keep important elements in center 66% circle

### favicon.png
- **Size**: 48x48px
- **Format**: PNG
- **Purpose**: Web favicon
- **Guidelines**: Simplified version of app icon

## Quick Start

1. Create or obtain your client's brand assets
2. Resize to specifications above
3. Save in this directory with exact filenames
4. Test with: `expo start`

## Tools

- [Figma](https://figma.com) - Design assets
- [ImageMagick](https://imagemagick.org) - Batch resize
- [TinyPNG](https://tinypng.com) - Compress PNGs

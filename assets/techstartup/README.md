# TechStartup Assets

Modern, gradient-based design assets for TechStartup.

## Required Files

### icon.png
- **Size**: 1024x1024px
- **Format**: PNG with transparency
- **Design**: Modern gradient logo with tech elements
- **Colors**: Indigo (#6366F1) to Purple (#8B5CF6) gradient

### splash.png
- **Size**: 1242x2436px
- **Format**: PNG
- **Design**: Centered logo on dark background
- **Background**: #0F172A (Dark slate)

### adaptive-icon.png
- **Size**: 1024x1024px
- **Format**: PNG with transparency
- **Design**: Simplified gradient icon
- **Safe Area**: Keep logo in center 66% circle

### favicon.png
- **Size**: 48x48px
- **Format**: PNG
- **Design**: Minimal tech icon

## Design Guidelines

TechStartup brand emphasizes:
- Modern gradients (Indigo → Purple → Pink)
- Clean, minimal design
- Dark mode friendly
- Tech-forward aesthetic
- Accessible color contrasts

## Color Palette

- Primary: #6366F1 (Indigo)
- Secondary: #8B5CF6 (Purple)
- Accent: #EC4899 (Pink)
- Background (Dark): #0F172A
- Background (Light): #F8FAFC

# ACME Assets

Place ACME-branded image files in this directory:

## Required Files

### icon.png
- **Size**: 1024x1024px
- **Format**: PNG with transparency
- **Brand**: ACME corporate logo
- **Colors**: ACME Blue (#1E3A8A) primary

### splash.png
- **Size**: 1242x2436px
- **Format**: PNG
- **Brand**: ACME splash screen with logo
- **Background**: #1E3A8A (ACME Blue)

### adaptive-icon.png
- **Size**: 1024x1024px
- **Format**: PNG with transparency
- **Brand**: ACME logo for Android
- **Colors**: Match icon.png

### favicon.png
- **Size**: 48x48px
- **Format**: PNG
- **Brand**: Simplified ACME icon

## ACME Brand Guidelines

- Primary: #1E3A8A (Blue)
- Secondary: #DC2626 (Red)
- Accent: #F59E0B (Gold)
- Typography: Corporate sans-serif
- Logo: Always use official ACME lockup

## Testing

Run with ACME config:
```bash
CLIENT=acme expo start
```
